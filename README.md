# RoadCity

A dark themed road travel website with scroll animations.

---

# What I built

I designed and coded everything myself — from the sticky scroll effect to the animated marquee that changes direction when you scroll up or down. Every section was a new challenge and a new thing I learned.

---



## Built With

- HTML, CSS, JavaScript
- GSAP + ScrollTrigger
- Lenis smooth scroll

---

## Features

- Sticky video scroll
- Animated marquee
- Horizontal scroll routes
- Interactive route explorer
- Masonry gallery
- Responsive design
